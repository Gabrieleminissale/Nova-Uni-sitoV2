# Nova Uni 
Sito vetrina del **Polo Nova Uni**, centro di orientamento universitario per le università telematiche del Gruppo Multiversity (Pegaso, Mercatorum, San Raffaele).

- **Dominio**: [novauni.it](https://novauni.it)
- **Stack**: HTML / CSS / JavaScript vanilla — nessun framework, nessuna build
- **Hosting**: GitHub Pages
- **DNS**: Aruba

---

## Struttura del progetto

```
novauni-sito/
├── index.html              homepage
├── css/style.css           tutti gli stili
├── js/main.js              navbar scroll, slider testimonianze, invio form
├── assets/                 logo + immagini
├── CNAME                   dominio per GitHub Pages
├── robots.txt              direttive per i crawler
├── sitemap.xml             mappa del sito per Google
└── README.md               questo file
```
